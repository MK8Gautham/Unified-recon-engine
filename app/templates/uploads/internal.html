Internal Data Upload

Upload History
Supported formats: CSV, Excel (XLSX/XLS). Maximum file size: 50MB.
Required Columns

Your file must contain these columns:

  • transaction_id - Unique transaction identifier
  • amount - Transaction amount

Optional columns: transaction_time, reference_id

Upload Instructions
  1. Prepare your internal transaction data file
  2. Ensure required columns are present
  3. Choose your file (CSV or Excel format)
  4. Click "Upload & Process" to begin
  5. The system will automatically parse and store the data
File Format Tips
  • Column names are case-sensitive
  • Ensure transaction_id values are unique
  • Amount should be numeric values
  • Date formats: YYYY-MM-DD or DD/MM/YYYY
  • Remove any header rows except column names
Sample Format
transaction_id amount transaction_time
TXN001 1000.00 2024-01-15 10:30:00
TXN002 2500.50 2024-01-15 11:45:00
